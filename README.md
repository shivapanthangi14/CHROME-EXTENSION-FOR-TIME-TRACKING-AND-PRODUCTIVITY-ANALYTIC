*DESCRIPTION*

This project focuses on the development of a Chrome Extension designed to monitor user activity on web browsers and provide meaningful insights into time usage and productivity. The main objective of this task is to help users understand how much time they spend on different websites and to classify their browsing behavior into productive and unproductive categories. This type of tool is highly useful for students, professionals, and remote workers who want to improve their time management and efficiency while working online.

The extension works by tracking the active browser tabs and calculating the time spent on each website. Whenever the user switches between tabs or loads a new page, the extension records the time spent on the previous site and stores it in local storage. The collected data is then processed and displayed in a user-friendly popup interface. The system also categorizes websites such as YouTube, Instagram, and Facebook as unproductive, while educational or work-related sites are considered as productive like Google,Chat gpt. This classification helps users quickly identify where they are spending most of their time.

In addition to basic tracking, the extension includes a productivity analytics dashboard. A bar chart is integrated using Chart.js to visually represent the time spent on different websites. This graphical representation makes it easier for users to analyze their browsing patterns at a glance. Furthermore, a weekly report feature has been implemented, which organizes the data date-wise and calculates the total time spent over a week. This helps users track their progress and make improvements over time.

For the development of this project, several tools and technologies were used. The extension was built using HTML, CSS, and JavaScript, which are the core technologies for web development. The Chrome Extension API was utilized to access browser tabs, manage background processes, and store user data. Chart.js was used for creating interactive and responsive charts. The development and coding were carried out using Visual Studio Code (VS Code), which is a popular and powerful code editor. Google Chrome browser was used for testing and debugging the extension through the developer mode.

This task has real-world applications in various domains. It can be used in educational environments to help students focus on studies, in corporate settings to monitor employee productivity, and in personal life to reduce time spent on distracting websites. It can also be extended further by integrating backend technologies such as Node.js and databases like MongoDB to store data permanently and provide advanced analytics.

In conclusion, this project demonstrates practical knowledge of browser extension development, real-time data tracking, and user interface design. It also highlights the importance of productivity tools in today’s digital world. The successful implementation of this extension showcases the ability to build a useful, real-world application using modern web technologies.
